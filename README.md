# 🏥 Multi Disease Prediction System

A Machine Learning based healthcare web application that predicts multiple diseases using user health data. This project uses Logistic Regression models and a Streamlit web interface for real-time prediction.

## 🚀 Features
- Diabetes Prediction
- Heart Disease Prediction
- Interactive Streamlit UI
- Real-time Prediction
- Confidence Score Display
- Machine Learning Model Integration

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Pickle

## 🤖 Machine Learning Algorithm
- Logistic Regression

## 📊 Workflow
1. Data Collection
2. Data Preprocessing
3. Handling Missing Values
4. Feature Scaling
5. Model Training
6. Prediction
7. Streamlit Deployment
